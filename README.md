# 🛒 Flipzon – E-commerce Web Application

## 🚀 Overview

**Flipzon** is a full-stack e-commerce web application built using **Spring Boot, Thymeleaf, and MySQL**. It provides a complete online shopping experience with separate modules for **Admin** and **Customer**.

---

## 🛠️ Tech Stack

* **Backend:** Spring Boot, Spring MVC, Spring Data JPA
* **Frontend:** Thymeleaf, HTML, CSS, JavaScript
* **Database:** MySQL
* **Cloud Storage:** Cloudinary
* **Payment Gateway:** Razorpay

---

## 🎯 Features

### 👨‍💼 Admin Module

* Admin login authentication
* Add, edit, and delete products
* Upload product images using Cloudinary
* Manage product catalog

---

### 👤 Customer Module

* User registration with email verification
* Secure login functionality
* Browse products with:

  * 🔍 Search
  * 🎯 Filters
  * 📄 Pagination
* Add/remove items from cart
* Smooth checkout experience

---

### 💳 Payment Integration

* Integrated **Razorpay** for secure online transactions
* Real-time payment processing

---

### ☁️ Image Handling

* Used **Cloudinary API** for:

  * Image upload
  * Storage
  * Fast delivery

---

## 🧠 Key Functionalities

* Session management for user authentication
* Dynamic UI rendering using Thymeleaf
* RESTful backend communication
* Real-time user interaction for seamless experience

## 🧪 Core Use Case Flow

1. User registers with email verification
2. Logs in and browses products
3. Adds items to cart
4. Proceeds to checkout
5. Completes payment via Razorpay
